# County Survey CRM

A lightweight single-page CRM for tracking county survey submissions through review stages.

## Features
- Local account creation and sign-in for your team.
- Survey dashboard grouped by workflow stage.
- Search and stage filtering by applicant, parcel, or survey type.
- One-click progression logic that automatically routes required surveys to County Surveyor Review.

## Getting started
Open `index.html` in a browser. Data is stored locally in your browser via `localStorage`.
